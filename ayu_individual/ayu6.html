Sample Calculation

Slides about the calculation

Given the data.

\(x_1\) \(x_2\) \(y\)
0 1 2 1
1 2 5 1
2 3 4 -1
3 4 0 -1
4 0 1 -1

Suppose we use this data to train an Adaboost model with the learning rate \(L=1\) and obtain the three stumps as follows.

  • Stump 1: \(I(x_1<2.5)\). Where \(I(\cdot)\) is an indicator function, taking value \(1\) if the condition inside the parenthesis is true or \(x_1<2.5\) and \(-1\) otherwise. This means that this stump will predict \(1\) for if \(x_1<2.5\) and \(-1\) otherwise.

  • Stump 2: \(I(x_2>1.5)\)

  • Stump 3: \(I(x_2>4.5)\)

  1. Compute the weight of the data in the first round.

  2. After the first round, which observations should have the weights increased? Which observations should have the weights decreased? Explain why.

  3. Compute the weights of the data in the second round and third round.

  4. Compute the voting powers of the three stumps. Which stump has the highest voting power?

  5. Draw the decision boundary of the adaboost. This video shows how to make decision boundary of the adaboost: link

    Class note for drawing decision boundary of Adaboost can be found here: Link
